## 🔧 How to install
Prepare your environnement
```bash
# Update
sudo apt update

# Install git
sudo apt install git -y
```

Start SABU Installation
```bash
# Clone the repositorie
git clone https://github.com/SABU-BSI/SABU.git
cd SABU

# Run the installation script
./install.sh
```
